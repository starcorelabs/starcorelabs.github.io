Linux Fail

I love Linux because it’s beautiful and open source. I have used some flavor of Ubuntu since 2008. Usually on a secondary PC or Laptop. Sometimes in VirtualBox. Unfortunately, I cannot replace my main Windows OS with Ubuntu. Because Linux Sucks!

I would love to give up Windows and use Ubuntu as my primary OS. Unfortunately, Linux fails me in many areas that are deal breakers.

Here’s a list of Linux issues that keep me from switching:

  1. Terminal - The Linux terminal rocks when I’m developing web applications. But I don’t like having to configure and program my OS to use the computer. I’d rather use it without having to touch the terminal unless I want to.

  2. Audio Issues - I use Google Hangouts / Google Voice, Facebook Audio, and Skype. The Mic I have (Samsung CU01) is recognized by Ubuntu but will not work out of the box. I’ve spent hours in the terminal configuring files and adjusting settings to force it to work. Sometimes it works for a short time but it eventually fails.

  3. Installing Apps - I like the Ubuntu App Store. However, not all the applications work with the latest and greatest version of Ubuntu. Some applications require using adding a private repository before installing through the terminal. I don’t want to waste two hours installing and configuring software. I want to use the software to accomplish other goals.

  4. RDP - I have a few computers and prefer using Remote Desktop to access them instead of physically getting up to work on them. Ubuntu to Windows RDP works well. Ubuntu to Ubuntu RDP is horrible. Most online resources suggest using VNC instead. VNC sucks compared to RDP.

  5. Shared Folders - Sharing folders across my local network under Windows is simple. Attempting to share folders under Ubuntu is painful. Requiring Samba, forces me to use the terminal to install and configure. Even then, it doesn’t easily allow me to set the correct permissions.

  6. Webcam - I use Skype and Facebook Video chat sometime. My Microsoft Webcam doesn’t work properly under Ubuntu. Of course, there are no Linux drivers for the webcam.

  7. Stability - Some releases of Ubuntu are not stable. Updating and upgrading the OS can cause a stable system to become unstable. I do not like it when my desktop and/or applications freeze or crash. It would be nice if the developers included warnings when updating and/or upgrading, that doing so may make my system unstable.

  8. Games - I don’t play games as much these days. However, standard Windows games that end with .exe or .msi do not work on Ubuntu. Sure, there are Steam games. Or I could run Windows games under Wine / Play-On-Linux. Still, trying to run games on Linux, like everything else, is 60% terminal configuration before you can load the software. After hours of work most games don’t run poorly and crash. There is no such thing as DirectX for Linux, because it is a library owned by Microsoft.

These are the things Windows does better than Linux. When Ubuntu fixes these issues, I’ll drop Windows and never look back. Until then, I’m stuck with Windows out of necessity.
