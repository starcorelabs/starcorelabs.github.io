What is UX Design?

UX Designers research how users interact with web and mobile applications to determine the optimal layout of graphics, text, and interactive components.

They use software like Photoshop, Illustrator, Balsamiq, or Figma to create wireframes, mockups, and prototypes. These graphics are used by developers as blueprints to build working web and mobile applications.

UX Designer's understanding of design principals and technical possibilities allows them to harmonize design and functionality. Providing users with an enjoyable experience.
