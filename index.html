About

I live in Indianapolis, Indiana. I have spent the last 5 years working in corporate environments as a Front-End Developer.

I use to write HTML, CSS/SCSS, and Bootstrap code to create mobile-friendly user interfaces for web applications.

Now my focus is on Graphic Design. I use Photoshop, Illustrator, Balsamiq, and Figma to create UX Designs for web applications as well as graphics for logos, marketing, t-shirts, book, and album covers.

My hobbies include improving my design skills through books, video courses, and hands on experiments.

I enjoy building file servers. As well as tinkering with various Linux desktop operating systems in VirtualBox.

When not geeking out on technology I can be found reading fantasy novels, comic books, playing video games like Magic the Gathering, Minecraft, Diablo III, or watching Comic Book, Comedy, or Sci-Fi movies.
