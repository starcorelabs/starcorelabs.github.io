Role

I worked as a UX/UI Designer at Aventiv on the AllPaid Portal project. I teamed up with the project manager and developers to design screens for open JIRA tickets. Together, we improved the user experience by turning requirements into clear, usable designs that helped move the project forward.

 

Challenge

The AllPaid Payment Portal helps local and state governments process online payments. When I joined the project, it had already been in development for over a year, with many new features planned on the roadmap.

The design system was already set. Colors, fonts, styles, and layouts were defined and approved. However, the previous designer could not keep up with the growing workload. This created a bottleneck that slowed development and increased costs.

The team needed a reliable UX/UI designer who could quickly review backlogged JIRA tickets and create clear, responsive screen designs. My role was to turn those planned features into ready-to-use designs so developers could move forward without delays.

Solution

To clear the backlog fast, I created a simple, repeatable design process using reusable templates and components. This helped speed up work while keeping everything consistent.

I worked closely with the project manager and used Trello to organize and track the most urgent JIRA tickets. This made it easy to see priorities and stay focused.

By collaborating with the project manager and developers, we completed the backlog and delivered key features, including:

  • AllPaid Wallet – Let users save up to five payment methods for quick checkout.

  • Shopping Cart – Allowed users to add multiple items and pay for everything at once.

  • Autopay – Gave users the option to set a default payment method for monthly payments.

This teamwork and structured process helped the team move faster, reduce delays, and deliver features on time.
