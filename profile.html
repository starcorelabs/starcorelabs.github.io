JavaScript is Sexy

Improving my JavaScript skills in preparation to learn Angular.js.

New

I've used JavaScript for the last couple of years for minor things like menus and validation.

With the rise of Node.js, Angular.js and other framworks I've been interested in digging deeper into the core language before trying to learn these new frameworks.

I can imagine that trying to troubleshoot code I'm not completely familiar with would create some unnecessary challenges. So I've decided to get more familiar with JavaScript.

I started learning with Lynda.com videos. As well as watching other videos. I've learned the basic building blocks of the language, but not how to put them together.

Then I stumbiled on to JavaScript Is Sexy and it changed everything. The way they layout what to learn and what order to learn it in makes more sense. Plus, I'm moving into code design patterns. Which are re-usable code structures designed to address a specific type of problem.

So far I'm really enjoying learning JavaScript. Once I move over to Angular.js I'm sure my creative will take over and I'll build some interesting things.
