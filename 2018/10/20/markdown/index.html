Markdown

2018-10-20

I love writing in Markdown. I’ve been using Markdown for over 5 years.
Markdown is a lightweight markup language with plain text formatting syntax.
Its best suited for documentation that doesn’t required overly complex formatting.

Since I already know HTML it only took me about 20 minutes to learn. Writing posts in Hexo, Change Logs, and Readme files on GitHub require Markdown.

I take notes in Markdown using Typora instead of online tools like Google Keep.
